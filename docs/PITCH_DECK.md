# Five-slide pitch deck copy

## 1 — CampusMemory

**A learning coach that remembers what matters—and lets students stay in control.**

Qwen Cloud Global AI Hackathon · MemoryAgent

## 2 — The problem

- Every new AI chat starts from zero.
- Students repeat goals, constraints, and skill level.
- Hidden or stale memory creates incorrect advice.
- Full-history prompting wastes tokens and exposes irrelevant data.

## 3 — The product

- Qwen extracts durable typed memories.
- New sessions retrieve only five relevant memories.
- Conflicting facts automatically supersede stale facts.
- TTL expiration and one-click forgetting keep the learner in control.
- Every answer includes a visible memory trace.

## 4 — Engineering depth

- Qwen structured JSON output.
- Stable-key conflict resolution with audit links.
- Explainable ranking: importance, relevance, and recency.
- Persistent H2 + Spring Boot + Docker on Alibaba Cloud ECS.
- Deterministic demo fallback and integration-tested lifecycle.

## 5 — Impact and next step

One reusable memory layer can personalize learning plans, interview preparation, and tutoring while keeping memory inspectable. Next: semantic retrieval, encryption, memory confirmation, and learning-outcome evaluation.
